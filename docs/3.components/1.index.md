# Components

nuxt-formisch auto-imports components from `@formisch/vue` with an `F` prefix.

::card-group
  ::card{icon="ph:article-duotone" title="FForm" to="/components/f-form"}
  Form wrapper with submit handling and validation.
  ::
  ::card{icon="ph:textbox-duotone" title="FField" to="/components/f-field"}
  Scoped slot component for field state access.
  ::
  ::card{icon="ph:list-plus-duotone" title="FFieldArray" to="/components/f-field-array"}
  Scoped slot component for array field management.
  ::
  ::card{icon="ph:magic-wand-duotone" title="FAutoForm" to="/auto-form/f-auto-form"}
  Generate forms automatically from Valibot schemas.
  ::
