# Auto Form

Auto Form generates forms automatically from Valibot schemas. Define your schema once and get a fully functional form with validation, proper input types, and customizable layout.

## Quick Start

```vue [components/ContactForm.vue]
<template>
  <FAutoForm :schema="schema" @submit="onSubmit">
    <template #actions="{ isDirty, isSubmitting }">
      <UButton type="submit" :loading="isSubmitting" :disabled="!isDirty">
        Submit
      </UButton>
    </template>
  </FAutoForm>
</template>

<script setup lang="ts">
import * as v from 'valibot'

const schema = v.object({
  name: v.pipe(v.string(), v.minLength(2)),
  email: v.pipe(v.string(), v.email()),
  age: v.pipe(v.number(), v.minValue(18)),
  subscribe: v.boolean()
})

function onSubmit(values: v.InferOutput<typeof schema>) {
  console.log(values)
}
</script>
```

This generates a form with:
- Text input for `name`
- Email input for `email`
- Number input for `age`
- Checkbox for `subscribe`
- Validation on all fields

## How It Works

1. **Schema Introspection** — Analyzes the Valibot schema to extract field metadata (type, constraints, required status)
2. **Field Mapping** — Maps schema types to appropriate input components
3. **Validation** — Uses the same schema for client-side validation
4. **Customization** — Override any field's appearance or behavior via `fieldConfig`

## Type Mapping

| Valibot Type | Component | Notes |
|--------------|-----------|-------|
| `string()` | `UInput` | Default text input |
| `string().email()` | `UInput[type=email]` | Detected from pipe |
| `string().url()` | `UInput[type=url]` | Detected from pipe |
| `number()` | `UInput[type=number]` | With min/max from constraints |
| `boolean()` | `UCheckbox` | — |
| `date()` | `UInput[type=date]` | — |
| `picklist()` | `USelect` | Options from picklist values |
| `enum()` | `USelect` | Options from enum values |
| `array()` | Field array | Repeatable fields |
| `object()` | Nested fields | Recursive introspection |
