# Composables

nuxt-formisch auto-imports composables from `@formisch/vue` for reactive form management.

::card-group
  ::card{icon="ph:form-duotone" title="useForm" to="/composables/use-form"}
  Create and manage form state with schema validation.
  ::
  ::card{icon="ph:textbox-duotone" title="useField" to="/composables/use-field"}
  Bind individual fields with reactive state and validation.
  ::
  ::card{icon="ph:list-plus-duotone" title="useFieldArray" to="/composables/use-field-array"}
  Manage dynamic arrays of fields with add/remove operations.
  ::
  ::card{icon="ph:rows-duotone" title="useFormFields" to="/composables/use-form-fields"}
  Helper to create fields with less boilerplate.
  ::
  ::card{icon="ph:link-duotone" title="useNuxtUIField" to="/composables/use-nuxt-ui-field"}
  Adapt field stores for Nuxt UI components.
  ::
::
