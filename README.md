Schema-based, headless form library for Nuxt.

## Quick Setup

```bash
pnpm add nuxt-formisch @formisch/vue valibot
```

```ts
// nuxt.config.ts
export default defineNuxtConfig({
  modules: ['nuxt-formisch']
})
```
